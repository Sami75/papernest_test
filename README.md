## Commande de setup projet (pour le projet react)
1-Afin d'installer le projet, vous devez lancer les commandes suivantes:
```shell
# Go to root of the app
cd compteur_react

```shell
# Install javascript dependencies (use yarn or npm)
yarn install

# Start local server
yarn start

```
2-Allez sur:
```shell
http://127.0.0.1:3000/

```
